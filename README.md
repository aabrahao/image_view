# image\_view

## A ROS2+Qt6 camera view

image\_view is a ROS2 node that subsiscribe to image\_raw from usb\_cam ROS2 node and the video using a QWidget.

## ROS2 usage

```bash
ros2 run usb\_cam usb\_cam\_exe
```

```bash
ros2 run image\_view image\_view
```
